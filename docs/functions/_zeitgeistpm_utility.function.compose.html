Function compose

  • compose<A, B>(ab: ((...a: A) => B)): ((...a: A) => B)

  • Compose functions.

    Example

    const add = (a: number) => (b: number) => b + a
    const sub = (a: number) => (b: number) => b - a

    const addTwoSubOne = compose(add(2), sub(1))

    addTwoSubOne(10) === 11
